Arquitetura de Dados

APIs e Monitoramento Espacial

💡
Nota de Arquitetura do Sistema (MVP): As APIs listadas nesta página representam o ecossistema real de dados que o AstroPharma utilizará em sua versão de produção. Elas possuem endpoints abertos e são perfeitamente integráveis para fornecer telemetria de chuva, umidade e temperatura.
Satélites Sentinel

Copernicus Data Space (ESA)

A Agência Espacial Europeia fornece a API OData do programa Copernicus. É a fonte principal para mapeamento de temperatura do solo e anomalias de umidade.

Endpoint Principal: /odata/v1/Products
Foco Epidemiológico: Dengue e Zika
Missão GPM

NASA POWER & GPM API

Projeto de Ciência Aberta da NASA que entrega dados meteorológicos em JSON. O sensor GPM é essencial para medir precipitação extrema (chuvas torrenciais).

Formato de Saída: JSON / NetCDF
Foco Epidemiológico: Cólera e Leptospirose
Dados Nacionais

INPE - CPTEC & BDQueimadas

O Instituto Nacional de Pesquisas Espaciais fornece os modelos climáticos locais do Brasil e o monitoramento em tempo real de focos de calor e fumaça.

Cobertura: Território Brasileiro
Foco Epidemiológico: Crises Respiratórias (Asma)
Resposta a Desastres

International Space Charter

Consórcio global que libera imagens de altíssima resolução imediatamente após grandes desastres naturais (enchentes, ciclones) para ajudar na logística emergencial.

Caso de Uso: Rotas Logísticas de Última Milha
Ativação: Sob demanda (Emergências)

Documentação de Integração (Swagger/OpenAPI)

Especificação técnica do Hub Preditivo exposto para o SUS e Indústria Farmacêutica (B2G/B2B)

openapi.yaml
openapi: 3.0.3
info:
  title: AstroPharma API - Documentação do Hub Preditivo
  description: >
    API B2B/B2G do AstroPharma. Consome telemetria geoespacial (ESA, NASA, INPE) 
    e expõe endpoints de alertas epidemiológicos e recomendações logísticas 
    preventivas para o SUS e Indústria Farmacêutica.
  version: 1.0.0
servers:
  - url: https://api.astropharma.com/v1
    description: Servidor de Produção (MVP)

paths:
  /alertas:
    get:
      summary: Retorna os alertas epidemiológicos ativos
      parameters:
        - name: regiao
          in: query
          required: false
          schema:
            type: string
        - name: risco
          in: query
          required: false
          schema:
            type: string
            enum: [ALTO, CRITICO, MODERADO]
      responses:
        '200':
          description: Lista de alertas recuperada com sucesso.
          content:
            application/json:
              schema:
                type: array
                items:
                  $ref: '#/components/schemas/AlertaEpidemiologico'

  /alertas/{id}/logistica-preventiva:
    get:
      summary: Sugere a ordem de transferência de estoque
      parameters:
        - name: id
          in: path
          required: true
          schema:
            type: string
      responses:
        '200':
          description: Recomendação gerada com sucesso.
          content:
            application/json:
              schema:
                $ref: '#/components/schemas/RecomendacaoLogistica'
